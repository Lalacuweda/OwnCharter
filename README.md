# OwnCharter: building in the open

OwnCharter is the AI co-founder for solo operators. You hand an agent real
authority to act inside your business, and every action it takes is governed at
the gate and provably yours. This is the live build log: one person building it,
dogfooding it on a real business every day, writing down what ships, what
breaks, and what is still wrong.

If you read the site and wondered whether any of it is real, this is where you
check. New entries land weekly.

## How to read this

- [CHANGELOG.md](CHANGELOG.md): every entry, newest first.
- [entries/](entries/): the full posts.

## Follow along

Watch this repo to get new entries, or be first into the alpha at
[owncharter.com](https://owncharter.com). First alpha targeted for late summer.
